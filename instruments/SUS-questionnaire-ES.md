# Escala de Usabilidad del Sistema (SUS) — versión en español

Esta es la versión en español del *System Usability Scale* de Brooke (1996) que fue administrada a los 12 tutores participantes del estudio piloto. La traducción fue preparada por el equipo de investigación y revisada por un profesional de la educación.

Cada ítem se responde en una escala Likert de 5 puntos: **1 = totalmente en desacuerdo**, **5 = totalmente de acuerdo**.

## Ítems

| # | Enunciado | Escala |
|---|-----------|--------|
| 1 | Me gustaría usar este sistema con frecuencia. | 1–5 |
| 2 | Considero que el sistema es innecesariamente complejo. | 1–5 |
| 3 | Considero que el sistema es fácil de usar. | 1–5 |
| 4 | Creo que necesitaría el apoyo de un técnico para poder usar este sistema. | 1–5 |
| 5 | Las distintas funciones del sistema están bien integradas. | 1–5 |
| 6 | Hay demasiada inconsistencia en el sistema. | 1–5 |
| 7 | Imagino que la mayoría de las personas aprendería a usar el sistema muy rápidamente. | 1–5 |
| 8 | Encontré el sistema muy engorroso de usar. | 1–5 |
| 9 | Me sentí muy seguro/a al usar el sistema. | 1–5 |
| 10 | Necesité aprender muchas cosas antes de poder empezar a utilizar el sistema. | 1–5 |
