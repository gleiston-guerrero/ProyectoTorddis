# `data/` — Pseudonymised experimental datasets

This directory contains the datasets that support the empirical results of the manuscript. The datasets are released under the Creative Commons Attribution 4.0 International (CC-BY 4.0) licence (see `../LICENSE-DOCS`).

## Structure

```
data/
├── data-dictionary.md         ← variable-by-variable description
├── raw/                       ← as collected, after pseudonymisation
│   ├── sus-responses.csv                    (12 guardians × 10 SUS items)
│   ├── likert-effectiveness.csv             (12 guardians × 7 effectiveness items + demographics)
│   ├── recognition-latencies.csv            (16 sessions × 4 AI modules)
│   └── open-ended-responses.csv             (12 guardians × 8 open questions, redacted)
├── processed/                 ← derived tables (from analysis notebooks)
│   ├── descriptive-statistics.csv
│   ├── normality-test-results.csv
│   ├── correlation-matrix.csv
│   └── anova-results.csv
└── README.md                  ← this file
```

## Pseudonymisation and redaction

Every dataset in `raw/` has been processed to remove any personally identifying information:

- Guardian names replaced by pseudonymous identifiers `G01`…`G12`.
- Exact ages of tutors bucketed to 5-year ranges.
- Free-text open-ended answers had proper names, place names, school names, workplace names and any other potentially identifying detail redacted and replaced with bracketed tokens (`[NAME]`, `[PLACE]`, `[SCHOOL]`, etc.).
- No image, video or audio of a participant is included in this repository.

The full protocol is documented in `../ethics/data-management-plan.md`.

## Reproducibility

Every figure and table of the manuscript's Results section can be regenerated by running the notebooks in `../analysis/` against the raw datasets in `raw/`. See `../analysis/README.md` for the exact mapping.

## Templates versus real data

The three CSV files that begin with a `# NOTE:` comment row are **placeholder templates**. The corresponding real (pseudonymised) values are being finalised by the study team and will replace the templates before the public release of version 1.0.0 of the reproducibility package on Zenodo. Once the real data are inserted, the placeholder comment row is to be removed.

The file `raw/recognition-latencies.csv` contains the real values, taken directly from Table 6 of the manuscript (no pseudonymisation was necessary because the sessions are labelled 1–16, without reference to individual participants).

## How to cite

If you use any dataset from this directory, please cite both the manuscript and the Zenodo DOI. See the root `../README.md` for the citation entries.
